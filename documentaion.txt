----------------STEPS TO DEPLOY THE CODE IN PRODUCTION-----------

1) Copy the changes from development branch to production branch

2) Minify the code by running "node minify.js" in root dir of dashboard

3) Push the changes in production branch

4) Login to jenkins: http://148.251.152.87:8081/jenkins/job/rocq_dashboard/

5) Press "Build Now" button

6) Congratulations! Your new code is live in production



-------------------MINIFICATION------------------
1)"node minify.js"  minifies and concatenates all the js, css files into two files rocq.css and rocq.js

2) To use the minified files, remove all the local js and css links from index.html and add rocq.css and rocq.js to it

3) index.html of Production branch already have the step 2 done.

------------------------DASHBOARD FLOW--------------------
1) /views/index.html is loaded by browser to load all the js files including angular js and its controllers and route.js

2) route.js contains $routeProvider which loads the appropriate html.

3) let say http://rocq.io/dashboard is opened. $routeProvider will call the welcome.html file to load.

4) welcome.html contains the sidebar on the left side and loads the approprite partial page on right side
